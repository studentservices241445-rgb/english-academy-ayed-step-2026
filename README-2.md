# موقع دورة STEP المكثفة 2026 — أكاديمية عايد

مشروع Static (HTML/CSS/JS) جاهز للنشر على GitHub Pages.

## تشغيل محليًا
افتح `index.html` مباشرة أو استخدم سيرفر محلي (مثل Live Server في VS Code).

## النشر على GitHub Pages
1) ارفع الملفات إلى مستودع GitHub.
2) Settings → Pages → اختر الفرع `main` والجذر `/`.
3) تأكد من وجود ملف `.nojekyll` لتفادي مشاكل المسارات.

## ملاحظة مهمة
- السعر والتسجيل يظهران فقط بعد إكمال اختبار تحديد المستوى (حسب المتطلبات).
- لا يوجد سيرفر أو قاعدة بيانات: البيانات الأساسية تُحفظ محليًا في المتصفح.
